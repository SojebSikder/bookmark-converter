## Description

Bookmark converter converts HTML bookmarks (Netscape format) to Markdown.

It's painfull to import browser bookmarks to notion. This tool converts your bookmarks to Markdown so you can easily import them into notion.

## Installation
Download the latest release from the [releases page](https://github.com/sojebsikder/bookmark-converter/releases).


## Usage

```bash
bc bookmarks.html bookmarks.md
```

## Usage as a Library
If you want to use this programatically, you can import it as a library:
```bash
go get github.com/sojebsikder/bookmark-converter
```

Check code usage in `cmd/main.go`
